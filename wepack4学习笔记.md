## 使用-webpack4-配置-TypeScript-开发环境

- [点击我跳转 使用-webpack4-配置-TypeScript](https://www.gowhich.com/blog/840)

```bash
# 创建文件
mkdir webpack4-typescript-demo

# 进入文件
cd webpack4-typescript-demo

# 初始化配置
npm init -y

# 安装 webpack-cli
npm install webpack webpack-cli --save-dev
```

> 项目结构如下

```
├── package.json
├── src
│   └── index.ts
└── webpack.config.js
```
