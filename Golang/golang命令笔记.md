## go.mod-的生成

```bash
export GO111MODULE=on

go mod init
```

## 设置-golang-下载代理

```bash
export GOPROXY=https://goproxy.io
```

## 打包命令-go-build

```bash
go build
```

## 调试命令-go-run

```bash
go run
```
